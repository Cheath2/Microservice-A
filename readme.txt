Book submission Microservice

Communication Pipe: ZMQ

Currently running on local port

    Actions:
     'submit_books' to submit new books
     'get_list' to get titles in user's data.

This Microservice uses a REQ-Reply ZMQ pattern to offer funcationality to update book information (title,author, url, and description) by username. A user can submit up to three books
